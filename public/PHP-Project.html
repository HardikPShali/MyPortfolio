PHP Project

Project Overview

RoomAndRide is a web Base Application. Admin can Maintains all the records of the Hotels and their Rooms, PG detail and Ride and Rider detail. Used for Provide facilities to book the Hotel, PG, Ride details of various kind of Areas & Relevant Rides .Takes care of Hotel, PG And Ride plan and Riders details.
It includes how the people will place their Ride to get advantage of his/her ride & services of his/her Ride. And User Can Search and Book others Rides, Hotel, PG From this Application.

Project Details

RoomAndRide provides Facilities to Visit and Find Hotels, PGs and Ride by an admin and Users.
It is useful to manage work for admin's the entire task like Hotel, PGs, Rides, and Users etc. from anywhere.
It will also help to keep all the data updated.
Registered User can Put Rides Detail for specific Journey plan that other users/Riders can get advantages for their travelling.
Users are provided the facility to see and maintain their Account information from anywhere.
Admin can login into admin side.
Admin can add, update, and delete, search Hotel, PGs, Rooms, Users details.
Admin can manage different Category and Subcategories for Hotels and their Rooms.
User can login to user's side.
User also can add, update, delete, search Ride details and others Riders who book their Rides.
User Also see Booking History of Hotel and PGs and Ride with payble amount.
